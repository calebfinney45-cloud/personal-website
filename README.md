# Personal Website

Lab: CSS Layouts & Responsive Grids
Student: Caleb Finney

Environment: WSL Ubuntu / VS Code

## Objective
To transition a static HTML structure into a responsive, modern UI using CSS Grid and Flexbox, while maintaining semantic integrity.

## Core Features
Intrinsically Responsive Grid: Used auto-fit and minmax(300px, 1fr) for the project section so it works on mobile without extra media queries.

Typography & Branding: Implemented Inter via @import for UI readability and Kaushan Script for personalized accents.

Color Palette: Alice Blue (#F0F4F8) and soft Lavender (rgba(93, 95, 239, 0.05)).
